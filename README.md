# humidity-monitor
A raspberry pi humidity monitor for my home.

## Install Dependencies

```bash
# Create database
sqlite3 homesensor.db

# Set Python 3 as default
sudo update-alternatives --install /usr/bin/python python /usr/bin/python3.5
```
